***************************************************************************************************
Vivado Instructions
***************************************************************************************************

To build vivado project enter the main directory and call
vivado -source top_design.tcl from command line.
	
If you've made changes to the block diagram that you want to save and push, regenerate the tcl 
scripts for the block diagrams using:
	* open block diagrams
	* enter into the tcl console "write_db_tcl mb_subsystem.tcl"
	
If you need to install the basys 3 package:
	Open Vivado
	Tools->XHub_Stores
	Find Basys3, right click, select install
	
If needed add C/Xilinx/Vivado/bin to system PATH variable to enable vivado execution from the command line

***************************************************************************************************
SDK Instructions
***************************************************************************************************
Launch Xilinx SDK 2018.2. You can do this from vivado using FIle->Launch SDK or use the start menu

Select File->Open Project from file system

Click directory, then select the CableCamFPGA.sdk folder. In the box below make sure that the following
three folders are found:
	cablecamSW
	standalone_bsp_0
	top_level_hw_platform_0
	
Click finish and the complete sowftware project shouldshowup. Look under cablecamSW/src to find 
main.cpp and other software files.
